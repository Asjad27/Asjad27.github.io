Cellular Network Signal Strength Mapping on Android

Description

  • Asjad Asif Jah

  • July 2015

Developed an android application based on crowdsourcing which records the current signal strength of the cellular network in use and relates it with the current GPS location. A heatmap is generated to visualize the data already collected. The data can be visualized for all the cellular networks together or for a particular network selected by the user.
